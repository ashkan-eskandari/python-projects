Dangerous Writing App

Dive into the Dangerous Writing App, a project inspired by Squibler.io. Crafted with Python, JavaScript, and Tkinter, this application introduces a unique challenge: the timer starts ticking only when the user stops writing. After 10 seconds, the screen is swiftly cleared, encouraging users to maintain an uninterrupted flow of thought. Experience focused and dynamic writing with the Dangerous Writing App.

Weather

The IP-Based Weather App combines Python, JavaScript, and Flask API to provide real-time weather conditions based on the user's IP location. Stay informed about current weather conditions effortlessly with the IP-Based Weather App.

Text To Speech

The PDF-to-Speech Converter empowers users to transform uploaded PDF files into spoken narratives. Using Python, JavaScript, and PyPDF, the platform seamlessly extracts text content from PDFs and employs the UnrealSpeech API for high-quality text-to-speech conversion. Experience accessibility as PDFs come to life through audible narration with this intuitive and dynamic converter.
